# 🤖 Twisto - AI Voice Assistant

Twisto is a **desktop-based AI voice assistant** built with **Python, Tkinter, and Speech Recognition**.  
It can listen to your voice, respond naturally, and perform useful tasks like opening apps, browsing the web, and more.

---

## ✨ Features
- 🎤 Voice recognition (Google Speech API)
- 🗣️ Text-to-Speech responses
- 🖥️ Beautiful chat-style GUI (Tkinter + ttkbootstrap)
- 🚀 Supports:
  - Open Notepad, Calculator
  - Open Chrome, WhatsApp
  - Play Music folder
  - Open Documents folder
  - Open YouTube
  - Tell current time
  - Exit the program
- 🔔 Desktop notifications
